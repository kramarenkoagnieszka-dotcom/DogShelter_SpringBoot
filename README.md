# Dog Shelter System — Spring Boot Edition 🐾

### 🚀 Project Overview
This is a modern, web-based evolution of the original **Dog Shelter Management System** (initially a Java Console Application). The project is currently being **migrated to a RESTful API** powered by Spring Boot.

The **primary goal** of this project is to demonstrate ability to work with the **Spring Boot ecosystem** and modern Java development tools. It serves as a comprehensive learning journey, documenting the transition from a Java Core console application to a professional, full-stack enterprise architecture.

### 🏗️ Tech Stack
* **Framework:** Spring Boot 3.x
* **Language:** Java 21
* **Build Tool:** Maven
* **Database:** PostgreSQL (Planned)
* **ORM:** Hibernate / Spring Data JPA (Planned)
* **API:** REST (JSON)
* **Testing:** JUnit 5, AssertJ

### 🛠️ Roadmap & Development Phases (Work In Progress)

The project is being migrated and developed in incremental stages to ensure a clean transition from the legacy console application to a modern web architecture:

1.  **Phase 1: Model Refactoring & Domain Design** * Migrating core data structures to a Spring-compliant format.
    * Refactoring identifiers and establishing proper object-oriented relationships to prepare for database persistence.

2.  **Phase 2: Service Layer & Business Logic** * Implementation of dedicated Services to handle core business rules.
    * Transitioning from procedural logic to Dependency Injection (DI) and Spring-managed components.

3.  **Phase 3: Persistence Layer & Database Integration** * Integration with **PostgreSQL** for reliable data storage.
    * Implementing **Spring Data JPA** to replace legacy file-based persistence.

4.  **Phase 4: Web Layer & REST API** * Developing RESTful controllers to expose system functionalities via JSON endpoints.
    * Handling request mapping and polymorphic data structures.

5.  **Phase 5: Security & Access Control** * Implementing **Spring Security** to manage authentication and Role-Based Access Control (RBAC).
